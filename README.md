Intelligente Systeme - Programming Assignments

Welcome to my repository! This portfolio contains a collection of programming assignments (PAs) completed for the **Intelligente Systeme (Intelligent Systems)** course at **Otto von Guericke University Magdeburg (OVGU)**. 

The primary focus of these projects is to understand the mathematical foundations of Machine Learning and Artificial Intelligence algorithms by implementing them directly from scratch, without relying on high-level ML libraries.

## Tech Stack & Tools
* **Language:** Python
* **Libraries:** NumPy (Used extensively for matrix operations and mathematical computations), Math
* **Concepts:** Machine Learning, Deep Learning basics, Optimization, Neural Networks, Evolutionary Algorithms,Learning Classifier Systems(LCS)
